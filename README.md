## Exploratory Data Analysis (EDA) using Tips Dataset

### 📌 Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Tips Dataset using Python libraries such as Pandas, Matplotlib, and Seaborn. The analysis helps identify patterns, trends, correlations, and outliers in the dataset through various visualizations.

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset Used

* Tips Dataset

The dataset contains restaurant bill information including:

* Total bill
* Tip amount
* Gender
* Smoking status
* Day
* Time
* Size of group

---

## 📊 Tasks Performed

* Loaded and explored the dataset
* Performed Univariate Analysis
* Performed Bivariate Analysis
* Generated Histograms and Boxplots
* Created Scatterplots and Barplots
* Generated Correlation Heatmap
* Used Pairplot for multivariable visualization
* Identified patterns, trends, and outliers

---

## 📈 Visualizations Used

* Histogram
* Boxplot
* Scatterplot
* Barplot
* Heatmap
* Pairplot

---

## 🔍 Key Insights

* Higher total bills generally resulted in higher tips
* Weekend days showed higher average bill amounts
* Positive correlation exists between total bill and tip
* Some outliers were identified in total bill values

Successfully performed Exploratory Data Analysis on the Tips Dataset and extracted meaningful insights using data visualization techniques.

The dataset is now better understood and ready for Machine Learning applications.
